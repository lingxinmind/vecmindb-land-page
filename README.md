# VecminDB 落地页

VecminDB 官方品牌站点与候选名单收集页。

## 技术栈

纯静态页面，零构建工具：

- **index.html** — 主落地页（i18n 双语、深色科技风）
- **pricing.html** — 定价页（自带 Early Bird 定价切换）
- **style.css** — 全局样式（Aurora 流光 / 玻璃拟态）
- **script.js** — 交互逻辑（语言切换、表单提交、动画、Telemetry）

## 本地预览

直接用浏览器打开 `index.html`，或：

```bash
npx serve .
```

## 部署

静态文件托管（GitHub Pages / Vercel / Nginx 均可），无构建步骤。
